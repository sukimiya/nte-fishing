# nte-fishing

《异环》（NTE）全自动钓鱼辅助工具。游戏可在后台运行，工具自动完成抛竿→等待上钩→钓鱼小游戏→关闭结算界面的完整循环。

## 功能

- 后台截图，无需游戏窗口置于前台
- 自动检测游戏状态（空闲 / 上钩 / 钓鱼小游戏）
- 钓鱼小游戏中实时控制玩家竖线跟随鱼的滑块
- 自动关闭结算界面并开始下一轮
- 系统托盘图标 + F12 热键开关

## 依赖

- Windows 10/11
- Python 3.11+（推荐通过 [uv](https://github.com/astral-sh/uv) 管理）
- 游戏分辨率建议 2560×1440 或 1920×1080

## 安装

```bash
git clone https://github.com/Lethewind/nte-fishing.git
cd nte-fishing
uv sync
```

## 使用

1. 启动游戏并进入可以钓鱼的场景
2. 运行辅助工具：

```bash
uv run main.py
```

3. 系统托盘出现图标后，切换到游戏窗口
4. 按 **F12** 启动自动钓鱼（再按一次暂停）
5. 退出：右键托盘图标 → 退出

## 配置

所有参数在 `config.py` 中，常用项：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `GAME_WINDOW_TITLE` | `"异环"` | 游戏窗口标题 |
| `TOGGLE_HOTKEY` | `"f12"` | 开关热键 |
| `DEAD_ZONE` | `10` | 控制死区（像素） |
| `END_WAIT_SEC` | `5.0` | 钓鱼结束后等待结算界面的时间（秒） |
| `MAX_WAIT_SEC` | `60.0` | 等待上钩超时后重新抛竿（秒） |

## 工作原理

```
IDLE → 按 F 抛竿 → 等待上钩
  ↓ 检测到蓝色上钩光晕
按 F 确认 → 进入钓鱼小游戏
  ↓ 持续控制 A/D 键，将玩家竖线保持在鱼的绿色滑块内
钓鱼结束 → 等待结算界面 → 按 ESC 关闭 → 回到 IDLE
```

**截图**：BitBlt 后台截图，不干扰游戏显示  
**状态检测**：OpenCV HSV 颜色过滤（鱼滑块绿色、玩家竖线黄色、上钩蓝色光晕）  
**按键注入**：F/ESC 通过 PostMessage 后台发送；A/D 通过驱动级注入（keyboard 库）

## 注意事项

- 需要以**管理员权限**运行才能向游戏进程发送按键
- A/D 键为全局注入，钓鱼小游戏期间（约 5-15 秒）避免在文本输入框中打字
- HSV 颜色阈值已针对默认画质校准，若画质设置不同可在 `config.py` 中调整
